# Quickstart: 简单登录功能

**Feature**: 001-simple-login
**Date**: 2025-02-26

## 快速开始

### 1. 登录系统

1. 访问应用 URL
2. 自动跳转到登录页面（如果未登录）
3. 输入任意用户名（非空）
4. 输入密码：`123456`
5. 点击"登录"按钮
6. 成功后跳转到仪表盘

### 2. 查看当前用户

登录成功后，页面顶部导航栏右侧显示当前登录用户名。

### 3. 退出登录

点击导航栏右侧的"退出登录"按钮，清除登录状态并返回登录页面。

## 测试场景

### 场景 1: 正常登录

```
用户名: admin
密码: 123456
结果: ✅ 登录成功，跳转到仪表盘
```

### 场景 2: 错误密码

```
用户名: admin
密码: wrong_password
结果: ❌ 显示"密码错误"
```

### 场景 3: 空用户名

```
用户名: (空)
密码: 123456
结果: ❌ 显示"请输入用户名"
```

### 场景 4: 状态持久化

```
1. 登录成功
2. 关闭浏览器
3. 重新打开浏览器，访问应用
结果: ✅ 保持登录状态，无需重新登录
```

### 场景 5: 路由保护

```
1. 未登录状态
2. 直接访问 /tickets
结果: ✅ 自动重定向到 /login
```

### 场景 6: 退出后访问

```
1. 登录成功
2. 点击退出登录
3. 尝试访问任意页面
结果: ✅ 被重定向到登录页面
```

## 密码说明

本系统为简单版本，统一密码为：**123456**

用户名可以是任意非空字符串，系统不做限制。

## 注意事项

1. **安全性**: 本功能为简单演示版本，不应用于生产环境
2. **密码**: 所有用户共享固定密码 123456
3. **持久化**: 使用 localStorage，清除浏览器数据会丢失登录状态
4. **无后端**: 认证完全在前端进行，无服务端验证
